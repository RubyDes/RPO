# Словарь

1. **Курс** — Структурированная образовательная программа, состоящая из уроков, материалов и тестов
2. **Студент** — Пользователь, который проходит курсы, сдает тесты и получает сертификаты
3. **Преподаватель** — Пользователь, который создает и редактирует курсы, проверяет задания, отслеживает успеваемость
4. **Администратор** — Пользователь, который управляет системой: пользователями, курсами, правами доступа
5. **Техническая поддержка** — Специалист, решающий технические проблемы и обновляющий систему
6. **Урок** — Отдельная единица курса, содержащая образовательные материалы (текст, видео, презентации)
7. **Тест** — Контрольное задание для проверки знаний студента, может содержать вопросы разных типов
8. **Сертификат** — Электронный документ, подтверждающий успешное завершение курса
9. **Прогресс обучения** — Статус прохождения курса студентом, включая пройденные уроки и результаты тестов
10. **Форум курса** — Площадка для общения студентов и преподавателей по темам курса
11. **Каталог курсов** — Структурированный список всех доступных курсов с фильтрами и поиском
12. **Аналитика** — Статистические данные по успеваемости студентов и эффективности курсов
13. **Уведомление** — Сообщение, отправляемое пользователю о важных событиях (новый урок, дедлайн теста, ответ на форуме)
14. **Рейтинг курса** — Оценка курса, выставленная студентами на основе их опыта прохождения
15. **Автоматизированная проверка** — Автоматическая оценка тестов без вмешательства преподавателя
